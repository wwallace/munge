# munge
GO version "Dirty little word munger" by Th3 S3cr3t Ag3nt (https://github.com/Th3S3cr3tAg3nt/Munge), written mostly by ChatGPT 

## Usage
```
  -input string
        input file
  -level int
        munge level [0-9] (default 5) (default 5)
  -output string
        output file
  -word string
        word to munge
```        
 ## Example
 ` .\main -input rockyou.txt -output rock-munge.txt`
